Department of Labor Redesign

Home / Projects /DoL

Project Details

We took the U.S. Department of Labor existing website and, using UI skills, redesigned it into a responsive and intuitive website.

(Disclaimer: This project was a class assignment and is not affiliated with DoL or any government agency.)

  • My Role: Project Manager, UI Designer and Researcher
  • Tools: Adobe Creative Cloud, InVision Studio, Sketch, Miro, Trello, Google Drive, Excel, PowerPoint

View Case Study

First Steps:

Because the DoL website existed, one of our first steps was determining elements that did and did not work.

This included an extensive heuristics evaluation, color testing, accessibility testing, and redline annotations.

Alongside this research, we created proto personas to get into the mindset, pains, and goals of potential users.

Usability Testing

After constructing a plan, we tested the current website with users during interviews.

This information turned into an affinity diagram that led to an empathy map and priority matrix to narrow down to 3 main aspects.

Constructing a New Sitemap

Once we updated our user to a user persona and defined our problem, we constructed a new hierarchy of information with various card sorting versions and a sitemap.

We also began a moodboard for inspiration and a style guide for consistency.

Lo-Fi Prototype

After many paper wireframes, we decided on which direction to take with a low-fidelity clickable prototype for both website and mobile versions.

With a testing plan focused on 3 main tasks, we set out for guerilla testing and 5-second testing.

Hi-Fi Prototypes...

We combined user research with technical research and created high-fidelity responsive web and mobile designs that were tested and iterated upon.

We maintained similar color psychology expected from a government website.

Alternate Designs...

Thinking outside of the primary U.S. government colors and design, and taking inspiration from state government websites, we created an alternative design.
